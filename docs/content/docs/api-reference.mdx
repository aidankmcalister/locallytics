## AnalyticsGrabber

The main React component that collects analytics data.

```tsx
import { AnalyticsGrabber } from "locallytics";

// Basic usage
<AnalyticsGrabber />;
```

## AnalyticsJSON

Fetch analytics data from your database.

```typescript
import { AnalyticsJSON } from "locallytics";

// Basic usage
const data = await AnalyticsJSON({
  startDate: "2023-01-01", // Optional start date (default: 30 days ago)
  endDate: "2023-12-31", // Optional end date (default: now)
  limit: 100, // Max number of events to return (default: 1000)
  offset: 0, // Pagination offset (default: 0)
});
```

## Event Tracking

Track custom events in your application:

```typescript
// Track a page view
window.locallytics.track("page_view", {
  path: window.location.pathname,
  referrer: document.referrer,
});

// Track a custom event
window.locallytics.track("button_click", {
  buttonId: "cta-button",
  section: "hero",
});
```

## Quick Reference

### Server-Side

```typescript
import { locallytics, AnalyticsJSON } from "locallytics";

// Initialize
const analytics = await locallytics({
  database: process.env.DATABASE_URL,
});

// Export handlers
export const { GET, POST } = analytics;

// Fetch metrics
const data = await AnalyticsJSON({});
```

### Client-Side

```tsx
import { AnalyticsGrabber } from "locallytics";

// Track pageviews automatically
<AnalyticsGrabber />;

// Track custom events
window.locallytics.track("event_name", { prop: "value" });
```
