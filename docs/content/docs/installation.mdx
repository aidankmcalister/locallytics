# Installation

Get started with Locallytics in your Next.js or React application.

## Install the Package

```bash
npm install locallytics
```

## Peer Dependencies

Locallytics requires the following peer dependencies:

- `react` >= 18
- `kysely` (if using database adapter)
- `pg` (if using Postgres)

Install them based on your database choice:

### For PostgreSQL

```bash
npm install kysely pg
```

### For SQLite

```bash
npm install kysely better-sqlite3
```

## Set Up Database

Create the analytics table using the appropriate schema for your database.

### PostgreSQL

```bash
psql $DATABASE_URL -f node_modules/locallytics/schemas/postgres.sql
```

Or manually run the schema from `node_modules/locallytics/schemas/postgres.sql`:

```sql
CREATE TABLE IF NOT EXISTS loc_events (
  id TEXT PRIMARY KEY,
  ts BIGINT NOT NULL,
  session_id TEXT NOT NULL,
  anon_id TEXT,
  type TEXT NOT NULL,
  name TEXT,
  url TEXT NOT NULL,
  path TEXT NOT NULL,
  referrer TEXT,
  title TEXT,
  props TEXT,
  screen_w INTEGER,
  screen_h INTEGER,
  ip_hash TEXT
);

CREATE INDEX IF NOT EXISTS loc_by_ts ON loc_events(ts);
CREATE INDEX IF NOT EXISTS loc_by_path_ts ON loc_events(path, ts);
CREATE INDEX IF NOT EXISTS loc_by_type_ts ON loc_events(type, ts);
```

### SQLite

```bash
sqlite3 your-db.sqlite < node_modules/locallytics/schemas/sqlite.sql
```

Or manually run the schema from `node_modules/locallytics/schemas/sqlite.sql`.

## Environment Variables

Add your database connection string to your environment variables:

```bash
DATABASE_URL=postgresql://user:password@localhost:5432/mydb
```
